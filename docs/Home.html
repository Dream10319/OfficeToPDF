Project Description

OfficeToPDF is a free command line tool that automates server-based PDF conversion for Microsoft Office 2007, 2010, 2013 and 2016 users.

It is distributed under an Apache 2.0 open source license and can be integrated into a batch conversion process or other applications to produce PDFs of the highest quality.

The tool requires an installation of Office 2007, 2010, 2013 or 2016 to work.

OfficeToPDF development is funded by Cognidox - Smart document management
